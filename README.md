# 🚀 Advanced MERN Todo Application

A production-ready, full-stack Todo application built from scratch. This project showcases a robust modern web development workflow, featuring advanced data filtering, pagination, analytical insights, and a polished UI.

## 🌟 Key Features

- **Full CRUD Operations:** Seamless task creation, retrieval, updates, and deletion.
- **Advanced Filtering & Sorting:** Filter tasks by completion status or creation timeframes.
- **Server-side Pagination:** Efficient data handling for large sets of tasks.
- **Analytics Dashboard:** Quick insights into task completion rates and productivity metrics.
- **Modern UI/UX:** Built with Tailwind CSS v4 and fully accessible component primitives via shadcn/ui.
- **Production Deployment:** Configured for seamless, free hosting.

---

## 🛠️ Tech Stack

### Backend

- **Node.js & Express:** Fast, unopinionated, minimalist web framework.
- **MongoDB & Mongoose:** NoSQL database for flexible data modeling.

### Frontend

- **React:** Declarative, component-based UI library.
- **Tailwind CSS v4:** Next-generation utility-first CSS framework for rapid styling.
- **shadcn/ui:** Beautifully designed, accessible components built on top of Radix UI.
